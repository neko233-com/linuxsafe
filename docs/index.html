🐧 Linuxsafe

Agent-first Linux security CLI. Automated threat detection, investigation, and remediation.

Quick Install

curl -sSL https://raw.githubusercontent.com/neko233-com/linuxsafe/main/install.sh | bash

Usage

# Scan for threats
linuxsafe scan /

# Detect system threats
linuxsafe detect

# View hardware info
linuxsafe hw

# Generate report
linuxsafe report

# Check status
linuxsafe status

Features

🔍 Threat Detection

Scan files for malware, rootkits, and suspicious patterns. Hash-based detection with EICAR support.

🕵️ Investigation

Deep forensic analysis of processes, files, users, and network connections.

🧹 Auto Cleanup

Automated remediation with dry-run preview and force confirmation safety.

📊 Reports

Generate JSON, HTML, or Markdown security reports with full findings.

🖥️ Hardware Info

System hardware detection: CPU, memory, disks, network interfaces.

🤖 Agent-First

JSON output by default. Deterministic exit codes for automation.
